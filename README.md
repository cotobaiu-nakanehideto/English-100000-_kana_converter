# コトバイウ呵名コンバーター（スタンドアローン版）

英文を、精密カナ発音記号「呵名（Cana）」および「IPA（国際音声記号）」に一括変換する自立型パッケージソフトウェアです。

## 呵名（Cana）について
カナの小字は子音である小さい音を、カナの大字は母音である大きい音を表しています。字の大きさと音の大きさを合わせたため、従来の日本語表記法と真逆の表記ルールになりましたが、直感的に外国語を話すことのできる最適なカナ発音記号です。アンダーラインは強勢アクセントを表します。ひらがなは本来の音の近似音、カタカナは日本語と同一の音ですので、注意すべき音とそうでない音の違いが一目瞭然となっています。「呵名」は、口偏（言葉や発音の意）に可能の可をあてがった漢字「呵」に由来し、英語の「Can（〜できる）」とも掛け合わせた造語です。

## コンバーターの特徴
完全自立型（スタンドアローン）: 外部サーバーとの通信を行わず、ブラウザ上に100万行規模の辞書データを高速に読み込んで安全に処理します。
Word連携最適化: 変換結果をMicrosoft Wordにコピー＆ペーストした際、フォントや段落構造が崩れないよう特殊なHTML/CSS設計が施されています。

## 使い方
1. 本リポジトリからフォルダごとダウンロード（またはクローン）します。
2. `kana_converter.html` をブラウザ（EdgeやChromeなど）で開きます。
3. 画面の指示に従い、同フォルダ内の `dictionary.csv` を読み込ませます。
4. 英文を入力し、「文章を変換」ボタンを押してください。
